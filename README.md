# React boilerplate

A simple react-webpack boilerplate using webpack 4, Babel 7 and Sass

## Getting started

1. Clone or download this repository
2. run `yarn install`
3. run `yarn start`
4. Go to http://localhost:8080 (or check console for the port) and start coding!
